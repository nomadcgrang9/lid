## 2. 프론트엔드 배포

### 방법 1: 자동 빌드 & 배포 (권장)

```bash
cd c:\PRODUCT\LID\frontend
npm run build
cd ..
firebase deploy --only hosting
```

### 방법 2: 수동 빌드가 안 되는 경우

프론트엔드 빌드가 멈추는 경우, 아래 단계를 따라주세요:

#### 2-1. 새 터미널 열기
Windows PowerShell 또는 CMD를 새로 열어주세요.

#### 2-2. 프론트엔드 빌드
```bash
cd c:\PRODUCT\LID\frontend
npm run build
```

**빌드가 완료되면 다음과 같은 메시지가 표시됩니다:**
```
✓ built in XXXms
dist/index.html                   X.XX kB
dist/assets/index-XXXXX.js        XXX.XX kB
```

#### 2-3. 빌드 완료 확인
```bash
dir dist
```

`dist` 폴더에 `index.html`, `assets` 폴더 등이 생성되었는지 확인합니다.

#### 2-4. Firebase 배포
```bash
cd ..
firebase deploy --only hosting
```

### 방법 3: 빌드 스크립트 직접 실행

아래 내용을 `build-and-deploy.bat` 파일로 저장하고 실행:

```batch
@echo off
echo Starting frontend build...
cd c:\PRODUCT\LID\frontend
call npm run build

if %ERRORLEVEL% NEQ 0 (
    echo Build failed!
    pause
    exit /b %ERRORLEVEL%
)

echo Build successful!
echo.
echo Starting Firebase deployment...
cd ..
call firebase deploy --only hosting

if %ERRORLEVEL% NEQ 0 (
    echo Deployment failed!
    pause
    exit /b %ERRORLEVEL%
)

echo Deployment successful!
pause
```

**실행 방법:**
1. 위 내용을 복사
2. `c:\PRODUCT\LID\build-and-deploy.bat` 파일 생성
3. 파일을 더블클릭하여 실행

## 4. 배포 후 확인 사항

### 4-1. Cloud Functions 확인
Firebase Console > Functions 메뉴에서:
- ✓ 11개 함수 모두 배포됨
- ✓ extractExcelContent 함수 확인
- ✓ extractImageContent 함수 확인
- ✓ generateArticle 함수 업데이트됨

### 4-2. 프론트엔드 확인
배포된 사이트에 접속하여:
- ✓ 관리자 페이지 > AI 글쓰기 지침 메뉴 확인
- ✓ 글쓰기 페이지 > Step 3 근거자료 업로드 UI 확인

### 4-3. 기능 테스트

#### 관리자 페이지 테스트
1. 관리자 페이지 접속
2. "AI 글쓰기 지침" 섹션으로 이동
3. 프롬프트 수정 버튼 클릭
4. `[근거자료내용]` 태그가 포함되어 있는지 확인
5. 수정 후 저장
6. Firestore에서 `settings/ai_prompts` 문서 확인

#### 근거자료 업로드 테스트
1. 글쓰기 페이지 접속
2. Step 1-2 작성 (주제, 사례)
3. Step 3 - 근거자료 섹션에서:
   - TXT 파일 업로드 테스트
   - Excel 파일 업로드 테스트
   - 이미지 파일 업로드 테스트
4. 글 생성 클릭
5. 생성된 글에 근거자료 내용이 반영되었는지 확인

## 5. 문제 해결

### 빌드가 멈추는 경우
- Ctrl+C로 중단 후 다시 시도
- 새 터미널 열어서 실행
- `npm cache clean --force` 후 재시도

### Cloud Functions 배포 실패 시
```bash
# 로그 확인
firebase functions:log

# 특정 함수만 배포
firebase deploy --only functions:extractExcelContent,functions:extractImageContent
```

### 권한 오류가 발생하는 경우
```bash
firebase login --reauth
```
