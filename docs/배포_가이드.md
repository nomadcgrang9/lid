# 배포 가이드

## 프로젝트 구조 (2025.03.05 통합 완료)

```
c:\PRODUCT\LID\
├── src/              # React 앱 소스
├── public/           # 정적 에셋
├── functions/        # Cloud Functions
├── dist/             # 빌드 결과물 (gitignore)
├── package.json      # 의존성 + 스크립트
├── vite.config.js    # Vite 설정
├── firebase.json     # Firebase 설정 (public: "dist")
├── .firebaserc       # Firebase 프로젝트
└── index.html        # 앱 진입점
```

> **이전 `frontend/` 분리 구조에서 루트 단일 구조로 통합됨.**
> 모든 명령은 루트(`c:\PRODUCT\LID`)에서 실행.

---

## 1. 프론트엔드 빌드 & 배포

### 한 방 배포 (권장)
```bash
cd c:\PRODUCT\LID
npm run deploy
```
= `npm run build && firebase deploy --only hosting`

### 빌드만
```bash
npm run build
```

### 배포만 (이미 빌드된 상태)
```bash
firebase deploy --only hosting
```

### Claude Code에서 빌드할 때
Git Bash(MSYS2) 환경에서 vite build가 멈출 수 있음. cmd.exe 경유:
```bash
cmd.exe //c "cd /d C:\PRODUCT\LID && npx vite build"
```

## 5. 문제 해결

### 빌드 멈춤 (Git Bash 환경)
`cmd.exe //c "npx vite build"` 로 실행

### Cloud Functions 배포 실패
```bash
firebase functions:log
firebase deploy --only functions:함수이름
```

### 권한 오류
```bash
firebase login --reauth
```

### 롤백
- Functions: Firebase Console > Functions > 이전 버전 복원
- Hosting: Firebase Console > Hosting > 릴리스 기록 > 이전 버전 복원
