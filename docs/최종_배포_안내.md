## ⚠️ 프론트엔드 빌드 문제

자동 빌드가 Vite의 rendering 단계에서 계속 멈추는 문제가 발생했습니다.

**원인**: Vite 빌드 프로세스가 비동기로 실행되면서 출력이 캡처되지 않는 문제로 추정됩니다.

## 해결 방법: 직접 실행

### 🔥 가장 확실한 방법

**새 PowerShell 창을 열고** 아래 명령어를 **하나씩** 실행하세요:

```powershell
cd c:\PRODUCT\LID\frontend
npm run build
```

빌드가 완료될 때까지 기다립니다 (1-3분 소요).

**빌드 성공 메시지 예시:**
```
✓ 1780 modules transformed.
rendering chunks...
computing gzip size...
dist/index.html                   4.23 kB │ gzip:  1.89 kB
dist/assets/index-abc123.css     45.67 kB │ gzip: 12.34 kB
dist/assets/index-def456.js     234.56 kB │ gzip: 78.90 kB
✓ built in 45.67s
```

빌드 완료 후:

```powershell
cd c:\PRODUCT\LID
firebase deploy --only hosting
```

## 대안 방법

### 방법 2: 배치 파일 사용

1. 메모장을 엽니다
2. 아래 내용을 복사하여 붙여넣기:

```batch
@echo off
cd c:\PRODUCT\LID\frontend
call npm run build
cd ..
call firebase deploy --only hosting
pause
```

3. `c:\PRODUCT\LID\배포.bat`으로 저장
4. 파일을 더블클릭하여 실행

## 문제 발생 시

### 빌드 오류
```powershell
cd c:\PRODUCT\LID\frontend
npm cache clean --force
npm install
npm run build
```

### Firebase 권한 오류
```powershell
firebase login --reauth
```

### 배포 실패
```powershell
firebase deploy --only hosting --debug
```
