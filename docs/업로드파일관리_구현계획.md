## 기능 명세

### 메인 화면 구성

```
┌─────────────────────────────────────────────────────────────────────┐
│  업로드 파일 관리                                                    │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                     │
│  [파일명 또는 작성자로 검색...]              [유형 ▼]  [최신순 ▼]    │
│                                                                     │
│  ┌──────────────────────────────────────────────────────────────┐  │
│  │ □   파일명                 유형        크기    날짜   올린사람│  │
│  ├──────────────────────────────────────────────────────────────┤  │
│  │ □   교육기본법.pdf         법령/지침   1.2MB   01/05   김OO   │  │
│  │ □   학교폭력예방법.pdf     법령/지침   2.4MB   01/04   이OO   │  │
│  │ □   2025교육통계.xlsx      통계자료    524KB   01/03   박OO   │  │
│  │ □   현장실태조사.pdf       기타        3.1MB   01/02   김OO   │  │
│  └──────────────────────────────────────────────────────────────┘  │
│                                                                     │
│  선택: 0개  [일괄 다운로드]  [선택 삭제]               < 1  2  3 >  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 동작 정의

| 동작 | 결과 |
|------|------|
| 행 더블클릭 | 해당 파일 1개 즉시 다운로드 |
| 체크박스 선택 + [일괄 다운로드] | 선택된 파일들 다운로드 |
| 체크박스 선택 + [선택 삭제] | 선택된 파일들 삭제 |
| [유형 ▼] | 드롭다운으로 파일 유형 필터링 (전체/법령지침/통계자료/기타) |
| [최신순 ▼] | 드롭다운으로 정렬 기준 변경 (최신순/오래된순/파일명순/크기순) |
| 검색창 입력 | 파일명 또는 올린사람 이름으로 검색 |

## 데이터 구조

### Firestore 컬렉션: `uploadedFiles`

```javascript
{
  id: "auto-generated",
  fileName: "교육기본법.pdf",
  fileType: "pdf",              // pdf, xlsx, csv, jpg, png 등
  category: "법령/지침",         // 법령/지침, 통계자료, 기타
  fileSize: 1258291,            // bytes
  storagePath: "uploads/xxx.pdf", // Firebase Storage 경로
  downloadUrl: "https://...",   // 다운로드 URL
  uploadedAt: Timestamp,
  uploadedBy: "작성자명",
  usedInArticles: ["articleId1", "articleId2"]  // 사용된 글 ID 배열 (선택)
}
```

### 파일 유형 분류 기준

| 업로드 위치 | category |
|-------------|----------|
| ③ 관련 법령/지침 PDF 업로드 (필수) | 법령/지침 |
| ④ 통계 및 근거자료 (선택) | 통계자료 또는 기타 |

## 참고: 현재 글쓰기 파일 업로드 구조

`StepInput.jsx`에서 업로드하는 파일:

```javascript
// 필수: 법령/지침 PDF
formData.pdfFiles: File[]
formData.pdfFileNames: string[]

// 선택: 통계/근거자료
formData.referenceFiles: File[]
formData.referenceFileNames: string[]
```

현재는 AI 분석용으로만 사용되고, 파일 자체는 별도 저장/관리되지 않음.
